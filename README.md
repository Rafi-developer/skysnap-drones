DESCRIPTION:
SkySnap is a responsive e-commerce website for selling drone cameras.
It features a modern UI with product listings, shopping cart functionality,
and contact form.

FEATURES IMPLEMENTED:
1. Responsive navigation with mobile hamburger menu
2. Dynamic product display with star ratings
3. Interactive shopping cart (add/remove items, calculate total)
4. Smooth scrolling navigation
5. Contact form with validation
6. Clean, student-friendly design

TECHNOLOGIES USED:
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
